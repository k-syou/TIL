# TIL (Today I Learn)

이 레퍼지토리는 오늘 하루 배운 것들을 정리하기 위한 공간입니다. 일자별로 폴더와 파일을 만들어 체계적으로 정리할 예정입니다.

## 기술별 폴더 구조

│   └── ...

## 작성 규칙

1. 매일 배운 내용을 기록합니다.
2. 각 주제는 별도의 파일로 작성합니다.
3. 파일명은 주제를 잘 나타낼 수 있도록 명확하게 작성합니다.
